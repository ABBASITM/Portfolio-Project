Car Sales By Location

This project is a car sale parsing program that utilizes various data structures to efficiently store and search through car data. The program uses a 2D hash graph to store car location data, with latitude and longitude coordinates as the keys. It also utilizes a hash table to store car data, with the car's make and model as the key. Additionally, the program uses a trie data structure to store and search through car data by country. Finally, it utilizes chained binary search trees (BSTs) to store and sort car data by various attributes such as age, price, and purchase date. Overall, the combination of these data structures allows for efficient storage and retrieval of car data, making it a useful tool for those in the automotive industry.
